# I.Ming-Mizuki
水木明体

由于一点字坊推出的一点明体我十分喜爱，但又因为大陆地区的汉字规范及我的个人习惯，所以我只能将其进行修改，望一点字坊可以谅解。

喜欢断笔、横捺，不喜欢避重捺风格的朋友可以选择。

注：常用字不包括繁体字。用于繁体字时可能会遇到部件不统一的问题。本字体是针对简体字而修改的，繁体字请直接使用一点明体原版，也请支持原版的劳动成果。

感谢一点字坊提供的一点明体！
修改人：岛田水木
